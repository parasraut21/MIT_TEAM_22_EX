# Exchange Rate Visualization Dashboard

This interactive dashboard allows users to visualize and compare exchange rates between different currencies against USD over various durations.

## Features

- Visualize exchange rate ratios between two currencies over time.
- Choose durations such as Year, Month, Week, and Quarter for comparison.
- Select specific years, months, weeks, or quarters for detailed insights.
- Comparison of quarterly trends available.

## Usage

1. **Install Dependencies**
   - Ensure you have installed the required Python packages (`pandas`, `matplotlib`, `ipywidgets`).

2. **Data Preparation**
   - Place your exchange rate dataset in CSV format.
   - Update the file path in the code to load your dataset (`exchange_rates = pd.read_csv("path/to/your/dataset.csv")`).

3. **Run the Dashboard**
   - Run the code in a Jupyter Notebook or JupyterLab environment.

4. **Interact with the Dashboard**
   - Use the dropdown menus to select currencies, durations, and specific time periods for visualization.
